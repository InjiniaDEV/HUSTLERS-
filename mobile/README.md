# HUSTLERS Mobile

React Native mobile app for the HUSTLERS platform.

## Tech Stack
- React Native (Expo or CLI)
- Redux Toolkit (State Management)
- Axios (API calls)
- React Navigation
- Secure Storage (for JWT)
- Image Picker (KYC uploads)

## Setup
1. Install Node.js (v18+ recommended)
2. Install Expo CLI: `npm install -g expo-cli` (if using Expo)
3. Clone this repository
4. Run `npm install`
5. Start the app: `npm start` or `expo start`

## Folder Structure
- src/
  - screens/
  - components/
  - navigation/
  - store/
  - utils/

## Key Features
- User onboarding & authentication
- KYC document capture & upload
- Profile management
- Secure token storage
